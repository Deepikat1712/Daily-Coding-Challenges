# Day 03 - LeetCode Practice

## Problem 1

### Smallest Even Multiple

- **Problem Number:** 2413
- **Platform:** LeetCode
- **Difficulty:** Easy
- **Language:** Java

### Concepts Learned

- Variables
- if-else Statements
- Modulus Operator (`%`)
- While Loop
- Return Statement

### Problem Summary

Given a positive integer `n`, return the smallest positive integer that is a multiple of both `n` and `2`.

### Key Learning

- Check whether a number is even using the modulus operator (`%`).
- Apply conditional logic using `if-else`.
- Understand how loops work to find a value.
- Learn that some problems can be solved with a simple mathematical observation instead of iteration.
